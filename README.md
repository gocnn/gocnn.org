# GoCNN's Website

This is the official website for GoCNN, built with Svelte.

## Developing

Once you've set up the project and installed dependencies with `bun install`, start a development server:

```sh
bun run dev

# or start the server and open the app in a new browser tab
bun run dev -- --open
```

## Building

To build the website for production:

```sh
bun run build
```

To serve the production build locally, you can run:

```sh
bun run preview
```
